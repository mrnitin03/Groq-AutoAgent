# DigitalNagari-Groq-AutoAgent

### Setup
1. Get Groq API Key from [Groq Console](https://console.groq.com/keys).
2. Generate Telegraph Token using `groq_token_generator.py`.
3. Add `GROQ_API_KEY` and `TELEGRAPH_TOKEN` to GitHub Repo Secrets.
4. Ensure Workflow Permissions are set to **Read and Write**.
