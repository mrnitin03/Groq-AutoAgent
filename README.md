# DigitalNagari-Groq-AutoAgent

Production-grade automation to generate SEO articles using Groq AI and publish them to Telegraph automatically.

## Setup Instructions

### 1. Get Groq API Key
1. Go to [Groq Console](https://console.groq.com/keys).
2. Create a new API Key.
3. Save it for later.

### 2. Generate Telegraph Token
1. Run the local generator:
   ```bash
